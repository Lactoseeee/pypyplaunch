## 📥 Installation

1. **Download the latest release**
   Go to [Releases](https://github.com/yourusername/helios/releases) and download `Helios_Setup.exe`

2. **Run the installer**
   Double-click `Helios_Setup.exe` and follow the setup wizard

3. **Allow permissions if prompted**
   Helios requires administrator rights to apply system tweaks — click **Yes** when Windows asks

4. **Launch Helios**
   Open Helios from your Desktop or Start Menu

5. **Click Boost & enjoy** 🚀
   Select a preset and hit **Apply** — that's it!

## ⚙️ How It Works

Helios applies a set of proven Windows optimizations at the system level:

- Switches power plan to **Ultimate Performance**
- Sets game process priority to **High** via Task Manager API
- Disables **Nagle's Algorithm** to reduce network latency
- Clears **standby RAM** before gaming sessions
- Disables **Xbox Game Bar**, Game DVR and unnecessary overlays
- Optimizes **GPU scheduling** (HAGS) via registry
- Disables unnecessary **background services** that compete for resources

All changes are **reversible** — use the Restore button to revert everything to default.
